### Code of SEEN

This package includes the Python code of SEEN Algorithm proposed in [1].

The `run_demo.py` function shows an example of using this code based on pendigits dataset.

[1]  Yong-Nan Zhu, and Yu-Feng Li. Semi-Supervised Streaming Learning with Emerging New Labels. In: The 34th AAAI Conference on Artificial Intelligence (AAAI'20), New York, New York, 2020.
